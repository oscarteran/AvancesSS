LISTADO DE COMANDOS PARA EL SHELL

* pwd
	Ruta absoluta del directorio actual
* cd
	Navegación entre directores: cd 'Destino'
	cd ..    Directorio superior
	cd 	 Directorio raíz
	cd - 	 Directorio anterior
* ls
	Muestra el contenido del directorio
	ls -R 	 Muestra archivos dentro de directorios internos
	ls -a 	 Muestra archivos ocultos 
	ls -al   Muestra archivos detallados
* cat 
	Lista el contenido en terminal de un archivo
* cp
	Copia archivos con el siguiente formato
	cp 'Archivo' 'Ruta destino'
* mv 
	Mueve archivos con el siguiente formato
	mv 'Archivo' 'Ruta destino'
* mkdir 	
	Crea un directorio nuevo
* rmdir 	
	Elimina un directorio vacío
* rm 	
	Elimina archivos y/o directorios no vacíos
* touch 
	Crea un archivo de cualquier tipo en algún directorio destino o en el actual
* locate
	Ubica un archivo y retorna el directorio
	locate -i palabra1*palabra2
* find 
	Buscador de archivos y directorios
* grep 
	Busca palabras a través de un archivo de texto
